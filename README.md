# draw3dBubblePlot
Draw 3D Bubble Plots


## Install
1. Install package devtools
2. Use *devtools::install()* to install this package

## 3D bubble plot sample
see *"sample.pdf"*

## Reproduce sample
please use *"sample.R"* and *"sample.Rdata"*
